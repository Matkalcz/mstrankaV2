Nový mcporter skill: Přímá integrace s MCP servery

OpenClaw 3.13 přináší revoluční novinku - mcporter skill, který umožňuje přímou komunikaci s MCP (Model Context Protocol) servery. Tato integrace otevírá zcela nové možnosti pro vaše AI agenty.

Co je MCP?

Model Context Protocol (MCP) je otevřený standard pro komunikaci mezi AI modely a externími nástroji. Umožňuje agentům přistupovat k databázím, API, souborovým systémům a dalším zdrojům.

mcporter skill v akci

Připojení k MCP serveru

mcporter config add mstranka \
  --transport http \
  --url https://mcp.v2.mstranka.cz/ \
  --api-key msk_...

Volání MCP nástrojů

mcporter call mstranka.list_websites

Vytváření obsahu

mcporter call mstranka.create_post \
  websiteId="..." \
  title="Můj článek" \
  content="$(cat obsah.html)"

Praktické využití

1. Správa webů přes mStranka V2

Váš agent může přímo:

  • Vytvářet a upravovat články
  • Spravovat kategorie
  • Nahrávat obrázky
  • Publikovat obsah

2. Automatizované reporty

Denní automatické reporty generované agentem a publikované na web:

openclaw cron add --name "daily-blog-post" \
  --schedule "0 8 * * *" \
  --command "agent run blog-agent --task 'Vytvoř denní shrnutí novinek'"

3. Multi-platform obsah

Jeden agent může publikovat obsah na více platformách současně:

  • Blog články na mStranka V2
  • Social media posty
  • Newsletter emaily
  • Dokumentaci na GitHub

Konfigurace mcporter

Instalace skillu

openclaw skill install mcporter

Konfigurace agenta

{
  "name": "Webový agent",
  "model": "gpt-4",
  "skills": ["mcporter"],
  "instructions": "Pomáháš s tvorbou a správou webového obsahu..."
}

Budoucnost MCP integrace

mcporter je jen začátek. Plánujeme:

  • Více MCP serverů out-of-the-box
  • Automatické objevování MCP nástrojů
  • GUI pro konfiguraci MCP připojení
  • Templates pro běžné workflow
