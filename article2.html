Začínáme s OpenClaw

OpenClaw je open-source platforma, která vám umožní vytvářet a spravovat osobní AI agenty. V tomto průvodci vás provedeme celým procesem od instalace po první automatizace.

1. Instalace

Požadavky

  • Node.js 18+
  • npm nebo yarn
  • Git
  • Přístup k terminálu

Instalace přes npm

npm install -g openclaw

Instalace přes Docker

docker run -d --name openclaw \
  -p 3000:3000 \
  -v /path/to/config:/app/config \
  openclaw/openclaw:latest

2. Konfigurace

První spuštění

openclaw init

Tento příkaz vytvoří konfigurační soubory a provede vás základním nastavením.

Klíčové konfigurace

  • API klíče - OpenAI, Anthropic, Google AI
  • Skills - výběr dovedností pro agenty
  • Workspace - nastavení pracovního adresáře
  • Agenti - konfigurace jednotlivých agentů

3. První agent

Vytvoření agenta

openclaw agent create muj-agent

Konfigurace agenta

Upravte soubor ~/.openclaw/agents/muj-agent.json:

{
  "name": "Můj první agent",
  "model": "gpt-4",
  "skills": ["web_search", "file_edit"],
  "instructions": "Jsi užitečný asistent..."
}

4. První automatizace

Příklad: Denní report

openclaw cron add --name "daily-report" \
  --schedule "0 9 * * *" \
  --command "agent run muj-agent --task 'Vytvoř denní report'"

5. Pokročilé funkce

  • MCP integrace - připojení k externím službám
  • Coding agent - pomoc s programováním
  • Webhooky - reakce na externí události
  • Multi-agent systémy - spolupráce více agentů

Více informací najdete v oficiální dokumentaci nebo na Discord komunitě.
