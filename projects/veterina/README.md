# Veterina Turnov - mStrankaV2 projekt

Tento projekt obsahuje design webu Veterina Turnov pro mStrankaV2 systém.

## Soubory:
- `veterina_header.html` - Navigační menu
- `veterina_footer.html` - Patička webu
- `veterina_styles.css` - CSS styly
- `veterina_homepage.html` - Ukázková homepage

## Zdroj:
Export z mStranka systému (záloha: veterina-backup-20260316-020753.json)

## Poznámky:
- Menu používá Bootstrap 3 třídy
- CSS obsahuje všechny potřebné styly
- Design je kompatibilní s mStrankaV2
